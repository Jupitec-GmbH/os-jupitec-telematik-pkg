## Installation

Add this repository to your OPNsense instance under **System → Firmware → Settings**:

```
https://jupitec-gmbh.github.io/os-jupitec-telematik-pkg
```

Then install via the **Plugins** tab or from the shell:

```sh
pkg install os-jupitec-telematik
```
